# keras_practice

## Reqirements
* Homebrew
* pyenv
* pyenv-virtualenv

## Instllation

* install anaconda3-5

```
pyenv install anaconda3-5.1.0
```

* create virtualenv

```
pyenv virtualenv anaconda3-5.1.0 keras-practice
```

* activate anaconda

```
pyenv local anaconda3-5.1.0/envs/keras-practice
```
